About the Project

Tuition Management Reimagined

EduTrack is not just another school app — it's a complete operations system for tuition centers. Teachers track attendance and enter marks, admins manage students, batches, and schedules, while students access personalized dashboards with report cards and performance analytics. Every feature is built for the messy reality of running a real coaching institute.

Core Features

What We Built

Every module is designed to eliminate manual tracking and bring structure to tuition operations.

Attendance Tracking

Daily attendance for every batch and subject. Teachers mark with a tap, admins see trends and reports instantly.

Report Card Generator

Auto-generated report cards with marks, grades, and teacher remarks — ready to print or share digitally.

Performance Analytics

Visual dashboards showing student progress over time — subject-wise, batch-wise, with trend analysis.

Fees Management

Track fees per student — paid, pending, overdue. Filter by batch, generate receipts, and send reminders.

Remarks System

Teachers leave personalized remarks for each student — behavioral notes, improvement suggestions, and praise.

Subject & Batch Control

Each batch has its own subject list. Admins configure subjects per batch, and teachers access only their assigned batches.

User Roles

Multi-Role Dashboards

Three distinct user roles, each with tailored access and controls.

Admin Dashboard

  • Manage all students & teachers
  • Create batches & subjects
  • View all reports & analytics
  • Fees tracking & reminders
  • Full system configuration

Teacher Dashboard

  • Mark daily attendance
  • Enter marks & grades
  • Leave student remarks
  • View batch performance
  • Access assigned batches only

Student Dashboard

  • View attendance history
  • Download report cards
  • Check fee status
  • Read teacher remarks
  • Track personal progress
